# AntGravity Reference-Aligned Visual Review

## 검증 일시
2026-03-21 16:32:27 KST

## 검증 범위
- `design/reference.png`의 시각 요소 추출이 구조 변경 없이 정리됐는지
- `research.md`, `docs/research.md`와 현재 구조 보존 원칙이 충돌하지 않는지
- 토큰이 실제 구현 시 "디자인만 변경"하는 가이드로 작동하는지

## 핵심 판정
레퍼런스는 대시보드 구조까지 포함한 이미지지만, 이번 문서의 올바른 해석은 구조 복제가 아니라 시각 요소 추출이다. 따라서 이번 리뷰 기준은 아래다.

1. 현재 구조 유지
2. 레퍼런스의 색, 선, 질감, 폰트 인상 반영
3. 캐릭터 감성 강화

## 접근성 메모
- 허니 골드 위 텍스트는 흰색보다 짙은 브라운이 더 안정적이다
- 본문은 항상 `--text-primary` 또는 `--text-secondary`를 사용한다
- 상태색은 수치와 보조 강조에만 쓰고, 긴 문장 전체에는 남용하지 않는다

## 구현 가드레일
- 레이아웃 자체를 레퍼런스처럼 갈아엎지 않는다
- 데스크톱 대시보드식 고정 사이드바를 새로 만들지 않는다
- 카드 위치와 컴포넌트 구조를 바꾸지 않는다
- 디자인 변경을 이유로 플로우를 건드리지 않는다

## 남아 있는 불확실성
1. 레퍼런스 원본 폰트 패밀리명은 확정할 수 없다
2. 원본 3D 개미 에셋 출처는 확인할 수 없다

그래서 문서에는 "정확한 이름"보다 "정확한 시각 특성"을 남겼다.

## 최종 판정
현재 구조를 유지하면서 레퍼런스의 색상, 테두리, 폰트 인상, 캐릭터 질감을 가져오려는 목적에는 적합한 문서 상태다.
