# 레퍼런스 기반 디자인 시스템 적용

## 작성시각
2026-03-21

## 해결하고자 한 문제
레퍼런스 이미지와 예시 코드를 기반으로 디자인 시스템을 완전히 재구성하고, 갈색 기반 색상, 테두리 추가, Lucide Icons 적용.

## 작업 내용

### 1. 디자인 규칙 업데이트

#### 색상 팔레트 변경

| 항목 | 기존 | 변경 (레퍼런스 기반) |
|------|------|----------------------|
| 배경 | #F5F0E6 | #F5F2ED |
| 주요 텍스트 | #333333 | #4A3728 (갈색) |
| 보조 텍스트 | #666666 | #A09080 |
| Primary | #F5C518 | #FFC107 (노란색) |
| 테두리 | #EBEBEB | #D1C7B7 |
| 호버 | #D4A912 | #5D4735 |

#### 테두리 스타일 추가

| 항목 | 기존 | 변경 |
|------|------|------|
| 카드 테두리 | 없음 | 1px solid #D1C7B7 |
| 라운드 | 12px | 16px (rounded-2xl) |

#### 아이콘 시스템 변경

| 항목 | 기존 | 변경 |
|------|------|------|
| 아이콘 | 이모지 | Lucide Icons (SVG) |
| 네비게이션 | 👑🔎📝 | crown, search, book-open |
| 검색 | 없음 | search |

### 2. 파일 업데이트

- design/MASTER.md: 디자인 시스템 마스터 문서 업데이트
- design/design-tokens.md: 토큰 정의 업데이트
- design/tokens.css: CSS 변수 업데이트
- src/styles/tokens.css: 실제 사용 토큰 업데이트
- index.html: 전체 UI 업데이트

### 3. Lucide Icons 적용

```html
<script src="https://unpkg.com/lucide@latest/dist/umd/lucide.min.js"></script>
<i data-lucide="crown" class="lucide-icon"></i>
```

사용된 아이콘:
- arrow-left: 뒤로가기
- search: 검색
- crown: 여왕의 지시
- book-open: 생존 일지
