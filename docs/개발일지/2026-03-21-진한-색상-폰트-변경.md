# 진한 색상 및 폰트 변경

## 작성시각
2026-03-21

## 해결하고자 한 문제
사용자 요청에 따라 테두리와 갈색을 더 진하게 변경하고, 가독성 좋은 무료 폰트로 변경.

## 작업 내용

### 1. 색상 변경 (레퍼런스 분석 기반)

| 항목 | 기존 | 변경 |
|------|------|------|
| 배경 | #F5F2ED | #F5F0E6 (레퍼런스와 일치) |
| Primary/버튼 | #FFC107 | #E6B873 (레퍼런스와 일치) |
| 텍스트 Primary | #4A3728 | #5D4037 (진한 갈색 유지) |
| 텍스트 Secondary | #A09080 | #8D6E63 (진한 갈색 유지) |
| 테두리 | #D1C7B7 | #8D6E63 (진한 갈색 유지) |

### 2. 폰트 변경 (가독성 중심)

| 항목 | 기존 | 변경 |
|------|------|------|
| 타이틀 | Jua (귀여운 폰트) | Noto Sans KR 800 |
| 본문 | Noto Sans KR 400-500 | Noto Sans KR 400-500 (유지) |
| 일기 | Nanum Pen Script | Nanum Pen Script (유지) |

### 3. 파일 업데이트

- design/MASTER.md: 색상 및 폰트 전략 업데이트
- design/design-tokens.md: 토큰 정의 업데이트
- design/tokens.css: CSS 변수 업데이트
- src/styles/tokens.css: 실제 사용 토큰 업데이트
- index.html: 전체 UI 색상 및 폰트 업데이트

## 해결된 것
- 레퍼런스 이미지와 배경색, Primary 색상 일치
- 테두리와 텍스트는 진한 갈색으로 변경
- Jua 폰트 제거, Noto Sans KR만 사용으로 가독성 향상
