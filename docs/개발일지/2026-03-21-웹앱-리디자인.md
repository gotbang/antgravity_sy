# 웹앱 리디자인 구현

## 작성시각
2026-03-21

## 해결하고자 한 문제
원본 HTML 소스코드의 형식과 내용 구조는 유지하면서 design 폴더의 새 디자인 시스템으로 시각적 디자인 전면 업그레이드

## 작업 내용

### 1. plan.md 작성
- 원본 vs 새 디자인 비교 분석
- 색상, 스타일 매핑 정리
- 컴포넌트별 변경 사항 정의
- 작업 순서 및 검증 체크리스트 작성

### 2. index.html 구현
- 로비 화면 (3개 모드 카드)
- 주식 앱 화면 (3개 탭)
  - Tab 1: 여왕의 지시 (오늘 판단)
  - Tab 2: 먹이 탐색 (종목 분석)
  - Tab 3: 생존 일지 (감정 일기)
- 캐릭터 말풍선 (병정개미, 여왕개미, 정찰개미)
- 하단 네비게이션 (인디케이터 포함)
- 화면 전환 애니메이션
- 탭 전환 로직
- 토스트 알림

### 3. 디자인 변경 사항

| 항목 | 원본 | 변경 |
|------|------|------|
| 배경색 | #F4F1EB | #FFF8E7 |
| 주요 텍스트 | #4A3525 | #333333 |
| 보조 텍스트 | #6B5D52 | #666666 |
| 강조색 | #D4C5B0 | #FF9900 |
| 카드 라운드 | 28px | 12px |
| 버튼 라운드 | 16px | 8-10px |
| 그림자 | 0px 4px 16px rgba(0,0,0,0.08) | 0px 4px rgba(0,0,0,0.1) |
