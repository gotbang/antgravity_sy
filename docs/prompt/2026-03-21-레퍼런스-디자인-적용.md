# 레퍼런스 디자인 적용 프롬프트

## 작성시각
2026-03-21

## 프롬프트 내용

### 1차 프롬프트
```
아이콘도 레퍼런스이미지처럼 교체해주고 캐릭터도 레퍼런스이미지로 교체. 그리고 테두리부분도 레퍼런스이미지처럼 해줘. 예시코드다 [React 코드]
교체하기전에 design폴더안의 디자인 규칙을 더 업데이트하고 코드구현중에 나한테 물어보고 확인받은후 구현
```

### 2차 프롬프트 (확인 질문)
```
색상 팔레트를 레퍼런스처럼 #4A3728(갈색) 기반으로 변경할까요? → 네, 갈색 기반으로 변경
캐릭터를 어떻게 변경할까요? → 병정개미, 여왕개미, 정찰개미는 유지하지만 이미지는 레퍼런스 같은 3d 귀여운 캐릭터
아이콘을 어떻게 변경할까요? → Lucide Icons 사용 (추천)
테두리 스타일을 어떻게 변경할까요? → 레퍼런스처럼 테두리 추가
```

### 3차 프롬프트 (추가 확인)
```
3D 캐릭터 이미지를 어떻게 처리할까요? → 이미지 업로드 대기
Lucide Icons를 어떻게 구현할까요? → Lucide CDN 사용 (추천)
```

## 사용된 스킬
- design-sample-master: 레퍼런스 이미지 분석

## 결과
- design/MASTER.md: 디자인 시스템 마스터 문서 업데이트
- design/design-tokens.md: 토큰 정의 업데이트
- design/tokens.css: CSS 변수 업데이트
- src/styles/tokens.css: 실제 사용 토큰 업데이트
- index.html: 전체 UI 업데이트
  - 갈색 기반 색상 (#4A3728, #F5F2ED, #D1C7B7)
  - 테두리 추가 (1px solid #D1C7B7)
  - Lucide Icons 적용
  - 라운드 16px

## 교훈
- 사용자에게 확인을 받으며 작업하면 방향성을 잃지 않음
- 레퍼런스 코드를 제공받으면 정확한 디자인 구현 가능
- Lucide CDN을 사용하면 별도 설치 없이 아이콘 사용 가능
- 캐릭터 이미지는 사용자 업로드 대기 상태로 두고 나중에 교체 가능
